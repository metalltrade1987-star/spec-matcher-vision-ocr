# Spec Matcher V2: Сверка спецификаций вентиляции через Vision AI

Инструмент для автоматической сверки коммерческих предложений (КП) с проектной документацией (PDF) в сфере вентиляции и автоматики. Использует локальную vision-модель Qwen2.5-VL 7B для OCR сканов спецификаций и гибридный fuzzy matching для сопоставления позиций.

##  Задача
Автоматизировать проверку соответствия оборудования в КП проектным спецификациям (187 страниц PDF), выявить расхождения в количествах и наименованиях.

## ️ Архитектура
Проект разделен на три независимых этапа:
1.  **Excel Parser** (`build_excel_tables.py`) — извлечение эталонных данных из КП, группировка и суммирование.
2.  **Vision OCR** (`parse_all_pdf.py`) — парсинг 47 страниц таблиц через Ollama + Qwen2.5-VL 7B на RTX 3060.
3.  **Hybrid Matcher** (`create_final_report_v2_safe.py`) — сопоставление через нормализацию артикулов + векторизованный fuzzy search (rapidfuzz).

##  Быстрый старт
```bash
# 1. Клонируй и создай окружение
git clone <url> && cd spec_matcher_v2
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt

# 2. Установи Ollama и модель
ollama pull qwen2.5vl:7b

# 3. Запусти пайплайн
python build_excel_tables.py
python parse_all_pdf.py      # ~25 мин на RTX 3060
python create_final_report_v2_safe.py
📊 Результаты
Обработано 47 страниц PDF за ~23 минуты.
Извлечено 597 позиций оборудования.
Автоматически сопоставлено 32 позиции (5% от общего объема).
Сгенерирован Excel-отчет с цветовой индикацией расхождений и листом unmatched для ручной проверки.
️ Ограничения
Локальная модель 7B имеет точность OCR ~60-70% для русского технического текста. Для production-использования рекомендуется замена на облачный API (GPT-4o/Claude) или более крупную локальную модель (32B+).
🛠️ Стек
Python 3.10, Pandas, OpenPyXL, RapidFuzz, pdf2image, Ollama, Qwen2.5-VL 7B, WSL2/Ubuntu.
